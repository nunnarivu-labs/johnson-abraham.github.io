# Johnson Abraham - Portfolio Website

A modern, responsive portfolio website showcasing my professional experience, projects, and technical skills as a Staff Software Engineer.

**Live Site:** [https://johnsonabraham.nunnarivulabs.in](https://johnsonabraham.nunnarivulabs.in)

## Tech Stack
- **Framework:** React 19 with TypeScript
- **Build Tool:** Vite (using rolldown-vite for faster builds)
- **Styling:** Tailwind CSS v4
- **Icons:** lucide-react

## Features
- Single-page application with smooth navigation
- Responsive design optimized for all screen sizes
- Dynamic experience calculation (auto-updates years of experience)
- Featured projects showcase with tech stacks and live links
- Professional timeline displaying work history
- Skills organized by category (Frontend, Backend, Cloud/Tools)
