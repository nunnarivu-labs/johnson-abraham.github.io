# AGENTS.md

## Project Overview
- Purpose: Personal website hosted from this repository.
- Stack: Vite + TypeScript + React.
- Source: `src/`
- Build output: `dist/`

## Build Commands
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Production build: `npm run build`
- Preview build: `npm run preview`
- Lint: `npm run lint`

## Coding Guidelines
- Prefer TypeScript for new code.
- Keep components small and focused; split by feature when they grow.
- Follow the existing formatting and linting rules.
- Avoid introducing new dependencies unless necessary.
- Update or add tests when changing behavior.
